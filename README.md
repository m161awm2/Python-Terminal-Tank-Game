# 🚀 Arc of Fate (운명의 탄도)

파이썬의 `curses` 라이브러리를 활용하여 제작한 **2인용 터미널 포트리스 게임**입니다. 고전적인 탄도학 계산을 바탕으로 다양한 아이템과 지뢰 시스템을 구현했습니다.



## ✨ 주요 기능

- **물리 엔진**: 중력($g=9.8$)과 각도, 파워를 반영한 실시간 탄도 계산.
- **아이템 시스템**:
  - 🏹 **Multi-Shot**: 7발의 포탄을 부채꼴 모양으로 동시 발사.
  - 🥎 **Bouncy**: 벽과 바닥에 튕기는 특수 포탄.
  - 💣 **Landmine**: 착탄 지점에 지뢰를 설치하여 밟을 시 대미지.
  - 🎯 **Full Guide**: 탄도 궤적을 끝까지 미리 보여주는 정밀 조준.
  - ⭐ **Trigger Drop**: 발사 후 `Space`를 한 번 더 누르면 탄이 수직 낙하하며 광범위 공격을 가하는 특수 아이템.
- **연료 시스템**: 이동 시 연료가 소모되어 전략적인 위치 선정이 중요합니다.
- **반응형 UI**: 터미널 크기에 맞춰 자동으로 지형과 플레이어 위치가 조정됩니다.

## 🕹️ 조작 방법

| 키 | 동작 |
| :--- | :--- |
| **방향키 좌/우** | 발사 각도 조절 |
| **방향키 상/하** | 발사 파워(세기) 조절 |
| **A / D** | 왼쪽 / 오른쪽으로 이동 (연료 소모) |
| **Space** | 포탄 발사 |

## 🐧 Linux 실행 방법

Arc of Fate는 터미널 기반 게임이므로 Linux에서 직접 실행할 수 있습니다.

### 1. Python 설치

```bash
sudo apt install python3 python3-curses
```

### 2. 프로젝트 다운로드

```bash
git clone https://github.com/YOUR_ID/Arc-of-Fate.git
cd Arc-of-Fate
```

### 3. 실행

```bash
python3 main.py
```
